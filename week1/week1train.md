# Que es MLOPS

MLOps tiene tres procesos:
1. Design,
2. Train
3. Operate

Tiene pipelines de entrenamineto, registro del modelo y despliegue.

En este curso vamos a entrenar un modelo que servira para predecir la distancia de viaje del trip dataset

2. Desviacion Estandar: 42.59
3. Dropping Outliers: 98%
